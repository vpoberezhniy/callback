Тестовое задание.
Рекомендации по установке:
1. Скачайте с Github репозиторий:  git clone https://github.com/vpoberezhniy/callback
2. После клонирования выполните в консоле команду: composer install
3. Создайте новую базу данных (MySQL) с именем "callback"
4. В консоле выполните команду: php artisan migrate
5. Далее в консоле запустим команду: php artisan db:seed
   (данная команда создаст ролии первого пользователя с ролью "manager")


!!! Важно !!!
Настройки файла .ENV Вы должны происать самостоятельно!
В даном проекте Вам необходимо прописать следующие настройки:
1) Подключение к БД (при написании проекта использовалось имя БД "callback")
2) Настроить блок SMTP (не забудьте прописать Ваш адрес на который буду поступать письма)

Развертывание готово.
